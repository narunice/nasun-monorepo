## 4. 리스크 관리

### 유동성 제공자 리스크

| 리스크 | 설명 | 완화 방안 |
|--------|------|----------|
| **Adverse Selection** | 정보 우위 트레이더에게 손실 | 스프레드 확대, 주문량 제한 |
| **Inventory Risk** | 한쪽 포지션 과다 보유 | 동적 스프레드 조정 |
| **Event Risk** | 갑작스런 뉴스로 급격한 가격 변동 | 주문 취소 자동화 |

### 시드 유동성 손실 시나리오

```
최악의 경우 (모든 포지션 손실):
- 시드 금액: 4,000 NUSDC
- 최대 손실: 4,000 NUSDC (테스트넷이므로 실제 손실 없음)

일반적인 경우:
- 스프레드 수익: 거래당 4%
- 예상 손익: 거래량에 따라 다름
```
